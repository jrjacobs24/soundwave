== Description ==

Blue Login Style is a tiny plugin which allows to customize your wp-login style easily with a click.<br />
Screenshots are available in <a target="_blank" href="http://preview.alimir.ir/wordpress-blue-login-style/">Here</a>

= Features =

*   Ability to choose nice styles with a click.
*   Ability to add message in your login page.
*   Add a custom logo of your wordPress blog as piece of cake.
*   Ability to set `login`, `logout` and `register` redirect links.
*   Simple configuration panel.
*   And so on...

== Installation ==

1. Upload `blue-login` files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Select your favorite style in `Blue Login Settings` panel.
4. Set your custom logo in `Select Login Logo` panel.
